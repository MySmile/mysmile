.. _glossary:

========
Glossary
========

.. glossary::

    additional menu
        Additional menu for dynamic navigation on inner pages. Number of elements of additional menu defined option MAX_INNERLINK_HISTORY
        
        .. seealso:: :ref:`UserConfiguration`.

    api page
        Page type **api page** is only available on API. 
        
        .. seealso:: :ref:`technical_configuration`.

    api & menu page
        This type combines the properties of **api page** and **menu page**.
        
        .. seealso:: :ref:`technical_configuration`.

    inner page
        Page, which is not accessible for users from the main menu called **inner page**. 
        Visited inner pages form a **additional menu**. 
        
        .. seealso:: :ref:`technical_configuration`.

    menu page
        Page, which is accessible for users from the main menu called **menu page**.
        
        .. seealso:: :ref:`technical_configuration`.
