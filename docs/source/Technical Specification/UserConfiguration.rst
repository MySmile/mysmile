.. _User_configuration:

User configuration
==================

User configuration in *config/local.py* or *config/production.py* contains general technical information such as:
    ::

      app_settings = {
	'PHONE': '000 000 000 00 00', 
	'EMAIL': 'myemail@email.com',
	'SKYPE': 'myskype',
	'GOOGLE_ANALITYCS_CODE': '',
	'MAX_INNERLINK_HISTORY': 4, 
	'REST_API': False
	}
